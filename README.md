# liveinconcert

Flutter/Dart mobile app to find concerts in Switzerland for artists you follow on Spotify.

## Technologies

- Dart/Flutter
- GraphQL API backend: https://github.com/fotcorn/liveinconcert-backend
- Firebase Cloud Messaging for notifications
